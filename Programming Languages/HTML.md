## Basics
- ```<button> blah blah </button>``` - adds button 'blah blah'
- ```<p> </p>``` - paragraph
- ```<h1> </h1>``` - heading (6 sizes h1 to h6)
- ```<body> </body>``` - everything else in the pagge
- ```<title> </title>``` - sets the page title shown on the browser tab
- ```<div> </div>``` - groups elements together

## Formatting
- ```<br>``` - a line break (self closing)
- ```<em> </em>``` - italic
- ```<strong> </strong>``` - bold

## Links & Attributes
- ```<a>href="insertlink"</a>``` - adds a link
- ```<img> src="insertlink" width="100" height="200"``` - adds an image
- ```<input>``` - gets input from user
- ```<input type = "text" placeholder = "Email">``` - placeholder attribute
- other input types: range, checkbox, date, color

## Lists in HTML
- ```<ol> </>``` - creates an ordered list
- ```<li> <//li>``` - adds items to the list
example:
```
<ol>
<li> Avengers </li>
<li> Avemgers 2 </li>
</ol>
```
- ```<ul> </ul>``` - creates an unordered list
example:
```
<!doctype html>
<html>
  <body>
    <ul>
      <li> Home </li>
      <li> About </li>
      <li> Contact </li>
    </ul>
  </body>
</html>
```
